OpenMFPOpen Micro Frontend Platform

OpenMFP enables the dynamic integration of services into a unified common interface experience via Micro Frontends

Get Started
GitHub

Micro Frontend Platform

A flexible Micro Frontend platform built on top of the Luigi Micro Frontend Framework, enabling the composition of technology-agnostic Micro Frontends into dynamic portals at runtime.

Easy to Extend

Seamless Micro Frontend integration through content configuration only, no coding required. Simply provide a content configuration file and register your Micro Frontend in your OpenMFP based portal.

Reusable Libraries

Reuse existing libraries and frameworks to build your Micro Frontends, reducing duplication and speeding up development. OpenMFP does not impose any restrictions on the technology you use.

Developer Experience

An intuitive, developer-friendly environment designed to make building Micro Frontends faster, smoother, and more enjoyable.
